### new additions in ES8

1) String padding - `.padStart()` and `.padEnd()`

```
'Pillow'.padStart(10); 
// "    Pillow" 10 spaces in total = string + spaces before string

'Pillow'.padEnd(10);
// "Pillow    "

'Pillow'.padEnd(10, '*');
// "Pillow****"
```
